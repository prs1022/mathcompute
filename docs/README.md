数值分析也称为计算方法，是计算数学的一个主要部
分.

    计算数学是数学科学的一个分支，主要研究用计算机求解各种数学问题的数值计算方法及其理论与软件实现. 

数值分析的内容包括函数的数值逼近、数值微分与数
值积分、非线性方程数值解、数值线性代数、常微和偏微
数值解等.
